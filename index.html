WordPress Installation

Before installing WordPress you need to have the server setup. As we use PHP and MySQL for WordPress we will go for installing:

  • WAMP for Windows
  • LAMP for Ubuntu
  • MAMP for Mac OSX

WAMP Installation

  • Download WAMP from wampserver.com.
  • Install using donwloaded exe file.
  • Confirm installation by opening http://localhost

LAMP Installation


sudo apt-get update

sudo apt-get install lamp-server^

sudo apt-get install phpmyadmin

sudo a2enmod rewrite && sudo service apache2 restart
  • Confirm installation by opening http://localhost

MAMP Installation

  • Download MAMP from mamp.info.
  • Install using donwloaded .pkg file.
  • Confirm installation by opening http://localhost

PHPMyAdmin

  • It is a tool to manage MySQL databases
  • Confirm its installation by opening http://localhost/phpmyadmin

htdocs/html directory

  • Main directory where your localhost server is pointed.
  • for Windows: C:\wamp\htdocs
  • for Ubuntu: /var/www/html
  • for MacOSX: /Applications/MAMP/htdocs

WordPress Installation

  • Download WordPress from https://wordpress.org/download/ or from link here.
  • Unzip the latest.zip in htdocs/html directory.
  • Now rename your wordpress folder to your sitename.
    e.g. mysite

Directory Structure

Make sure your directory structure should be like this:


htdocs or html/
└── mysite/
    ├── wp-admin/
    ├── wp-content/
    ├── wp-includes/
    ├── index.php
    ├── license.txt
    ├── readme.html
    ├── ...
    └── ...
